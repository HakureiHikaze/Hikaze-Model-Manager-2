# Hikaze Model Manager 2

Hikaze Model Manager 2 是一个 ComfyUI 扩展，提供独立的模型管理界面，并在节点中提供快捷的模型选择器。

## 你可以做什么
- 以搜索、标签过滤、卡片或列表视图浏览模型库。
- 查看模型细节，包括预览图、标签、提示词和描述信息。
- 从节点打开管理器，选择 Checkpoint 或 LoRA 列表并回写到节点。
- 审核待导入模型，并迁移到正式库。
- 默认过滤敏感内容，保证安全优先的浏览体验。

## 在哪里使用
- ComfyUI 节点内：Checkpoint Loader 和 LoRA Power Loader 可直接打开管理器。
- 独立的全屏管理器页面，便于集中浏览与编辑。
